# Data Sources & Attribution

The bundled JSON datasets are generated by the scripts in `/scripts`. Regenerate
with `npm run data:all`. There is no official JLPT vocabulary/kanji list (the
Japan Foundation stopped publishing them in 2010); these are community lists.

## Kanji — `data/n5-kanji.json`

- **Source:** [davidluzgouveia/kanji-data](https://github.com/davidluzgouveia/kanji-data),
  a static KANJIDIC-derived dataset.
- **N5 selection:** entries where `jlpt_old === 4` (the pre-2010 JLPT level 4,
  which corresponds to the ~103-character community N5 set).
- **Underlying data:** KANJIDIC2 / EDICT, © the Electronic Dictionary Research
  and Development Group (EDRDG), used under the EDRDG licence. On'yomi are
  converted to katakana and meanings lower-cased during normalization.

## Vocabulary — `data/n5-vocab.json`

- **Source:** [vitto4/MinnaNoDS](https://github.com/vitto4/MinnaNoDS)
  (`minna-no-ds.yaml`), the vocabulary of *Minna no Nihongo Shokyū Ⅰ & Ⅱ*. We
  take **Shokyū Ⅰ (lessons 1–25)** as the strict N5 set and keep each word's
  Minna lesson (`mnnLesson`) and volume (`mnnBook`).
- **⚠ Copyright / licence:** The word selection and translations are the
  property of **© 3A Corporation**. This dataset is **non-commercial** and is
  intended to be used **in conjunction with — not as a substitute for — the
  Minna no Nihongo textbooks**. This app bundles it for personal, non-commercial
  study only; do not use it commercially. The upstream copyright notice is
  preserved and must remain with any redistribution.
- **Fields:** `word` (kanji form, or kana when there is none), `reading` (kana),
  `romaji` (from the dataset), `meanings` (the dataset's English gloss),
  `mnnLesson`, `mnnBook`. `partOfSpeech` is a surface-form heuristic
  (`src/lib/data/pos.ts`) — approximate. `audioUrl` is null (Web Speech at
  runtime).

## Grammar — `data/n5-grammar.json`

- Hand-authored reference list of ~16 core N5 grammar points (particles, polite
  forms). Reference material for study, no drilling.

## Stroke order — KanjiVG

- **Source:** [KanjiVG](https://kanjivg.tagaini.net) SVG data, © Ulrich Apel,
  licensed **CC BY-SA 3.0**. Fetched on demand at runtime from the jsDelivr CDN
  (`cdn.jsdelivr.net/gh/KanjiVG/kanjivg`) by KanjiVG id (unicode codepoint) and
  cached by the service worker. Used for the "Learn to Write" stroke-order
  animations and tracing guides. Not bundled or modified.

## Audio

- v1 uses the on-device Web Speech API. `scripts/generate-polly-audio.ts` can
  pre-generate Amazon Polly clips to S3 and populate `audioUrl`; it is not run by
  default and requires AWS credentials.
